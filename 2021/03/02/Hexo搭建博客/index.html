安装

在安装Node.js和Git的基础上，使用npm进行Hexo的全局安装。

cd进入目标目录，完成项目的clone和初始化。

在blog目录下安装maupassant主题（莫伯桑），并安装相应依赖。

注：若npm install hexo-renderer-sass安装时报错，可能是国内网络问题，请尝试使用代理或者切换至淘宝NPM镜像安装

进入文件目录结构。

其中

  • _config.yml保存了网站的配置信息，可以在此配置大部分的参数。
  • scaffolds文件夹保存博客文章和页面结构模板。
  • source文件夹保存博客和页面的markdown文件。
  • theme保存主题文件。

配置

在_config.yml中配置网站基本信息

别忘在theme中配置maupassant主题

进入theme目录下的_config.yml文件配置主题相关的信息

主题提供了disqus、gitalk、valine、minivaline、utterances、twikoo等comment插件，可以在配置中修改相应的enable属性来开启/关闭。也可以开启Google，Baidu和站内搜索功能。

menu栏修改nav-bar的显示项目，并使用$ hexo new page <pagename>新建相应页面。在新建页面文件夹内写入index.md文件来获得页面样式。

其中icon名由FontAwsome提供。

widgets修改side-bar的显示项目
